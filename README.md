# todotxt-mode README

"todotxt-mode" is an extension for working with todo.txt formatted files. 

## Features

Highlighting of todo.txt formatted files
- Priority
- Context
- Project
- Tag

## Extension Settings

Default styles
- `todotxtmode.completedTaskStyle`
